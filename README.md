# Página web de cifrado y descifrado
Esta es una aplicación web sencilla, que forma parte del Challege One Oracle, permitiendo a los usuarios cifrar y descifrar información utilizando un algoritmo de cifrado personalizado. La página web está construida con HTML, CSS y JavaScript. 

# Demostración en vivo
Puedes probar la demostración en vivo de la página web visitando el siguiente enlace:
Enlace: https://mbarrerag.github.io/EncryptPage-ChallegeOneOracle

# Características
*Campo de entrada para introducir la información a cifrar o descifrar.
*Botón de cifrado para cifrar la información introducida.
*Botón de descifrado para descifrar la información cifrada.
*Visualización del resultado cifrado o descifrado en la página web.
*Algoritmo de cifrado
*El algoritmo de cifrado utilizado en esta página web es un simple cifrado de sustitución. Reemplaza cada carácter en la cadena de entrada con otro carácter basado en un mapeo predefinido. El proceso de descifrado es la inversa de este mapeo.

# Cómo usar
1. Abre la página web utilizando el enlace proporcionado.
2. Introduce la información que quieres cifrar o descifrar en el campo de entrada.
3. Haz clic en el botón "Cifrar" para cifrar la información o en el botón "Descifrar" para descifrar la información.
4. El resultado cifrado o descifrado se mostrará en la página web.
